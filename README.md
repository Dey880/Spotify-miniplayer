# schitzo rambling
Decided to make a miniplayer myself since i didnt really like the stock one and there werent any good alternatives :p. was using the starryNights Forest theme when developing so it mirrors the design, dont think any other themes would break, but i can't guarantee compatability.

currently not finished but its getting somewhere.


# Installation

put miniplayer.js in `%appdata%\Roaming\spicetify\Extensions` and run `spicetify config extension miniplayer.js` and then `spicetify apply` to apply. you will now have a new button on your top bar :þ enjoy!"
